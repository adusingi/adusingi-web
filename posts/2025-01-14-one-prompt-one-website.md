Last in November 2024, I joined Nippon IT Systems, and since then, I have been working as a software engineer, which was completely new to me. Prior to joining the company, I had been managing software development teams. I did not write code in my day-to-day work. Our company offers a SaaS product, and my role involves maintaining code and adding new features. We use PHP (specifically [CakePHP](https://cakephp.org/) framework).

During my years at engineering school and while managing software development teams, I acquired knowledge of how software works and became quite comfortable reading lines of code if I needed to understand a feature, etc.

After spending more than a year writing lines of code every day, I have gained more experience, and I wanted to build my own project from scratch so I can gain a deep understanding of main components of a web application (Authentication, Database, API, Routing, Payment, etc).

On November 18, 2025, Google released an AI tool called Antigravity IDE. Using Antigravity is like having a developer on call — you describe what you want, and it builds it for you.

Since Google is offering a one-month free trial, I decided to test it and to create my portfolio site: [adusingi.com](http://adusingi.com)

## My Step-by-Step Process

Here are the steps I followed, and these are the same steps I used for all my projects:

1. I opened Google's AI Studio. (I sometimes use [Lovable](https://lovable.dev/), [v0](https://v0.app/), [Replit](https://replit.com/))
2. I typed one request prompt: "Build me a minimalist portfolio website" + pasted my LinkedIn bio
3. The AI generated all the code
4. I connected to my GitHub account
5. I uploaded it to a free hosting service. For this project, I used [Vercel](https://vercel.com/), but I sometimes use [Netlify](https://www.netlify.com/) or [OVH](https://www.ovhcloud.com/)
6. I connected my domain name: adusingi.com
